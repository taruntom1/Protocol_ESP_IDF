Configuration structure for \doxylink{class_u_a_r_t_protocol}{UARTProtocol}. 

This structure holds the configuration parameters required to initialize the \doxylink{class_u_a_r_t_protocol}{UARTProtocol} class, such as baud rate, UART port, buffer size, and pin settings.
